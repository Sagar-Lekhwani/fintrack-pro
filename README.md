# 💰 FinTrack Pro – Personal Finance Tracker

This repository contains the successful completion of the **FinTrack Pro Assignment** provided in the **Cohort 3.0 Batch**.

The objective of this project was to build a fully functional **Personal Finance Tracker** that helps users manage their income, expenses, and financial records with a clean and interactive user interface.

Unlike previous UI recreation assignments, this project focuses on implementing real-world functionality using **HTML, CSS, JavaScript, and Local Storage**.

This project is another milestone in my journey of mastering frontend development from **Basics → Advanced** through hands-on projects.

---

# 📌 About The Project

**FinTrack Pro** is a personal finance management application that allows users to keep track of their daily financial activities while storing data locally in the browser.

The project focuses on both **UI design** and **JavaScript-based functionality**, making it a practical frontend application.

---

# ✨ Features

- 👤 User Profile Management
- 💸 Add Income & Expenses
- 📊 Dashboard with Financial Summary
- 📜 Transaction History
- 🔍 Search & Filter Transactions
- ✏️ Edit & Delete Transactions
- 📂 Category-wise Organization
- 💾 Local Storage Integration
- 📱 Responsive User Interface
- 🌙 Modern and Clean Dashboard Design
